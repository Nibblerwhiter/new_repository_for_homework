Всем читающим привет! Меня зовут Александр Александрович Губатенко. Помимо увлечённого прохождения курса, я занимаюсь написанием стихов.

Вот один из них:

Приумножай свою реальность, 
Дополни будничный контраст!
Дари невеждам бурю, радость,
Дари ханжам их ветхий наст!
 
Забудь банальность - черт бы с ней,
Зачем тебе повторы партий, 
Когда в  душе твоей горят
Осколки прожитых страстей!
